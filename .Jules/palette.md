## 2026-03-26 - [Interactive Nested Elements & Redundant Screen Reader Announcements]
**Learning:** In interactive elements nested inside a larger clickable container (e.g., a "wishlist" button inside a clickable `ProductCard`), clicking the nested element also triggers the container's `onClick` unless explicit prevention is used. Also, visual rating systems (like star icons) typically cause redundant, noisy announcements for screen readers if paired with text containing the rating, as each star image has an implicit role if lacking `alt=""` and `aria-hidden`.
**Action:** Always use `onClick={(e) => e.stopPropagation()}` on interactive elements nested within clickable containers to prevent unintended navigation. Hide visual rating icons from screen readers using `aria-hidden="true"` on the wrapper and `alt=""` on individual images if the text rating is already visible or provided elsewhere.
