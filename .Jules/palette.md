## 2024-04-07 - Preventing Unintended Navigation and Redundant Screen Reader Announcements in Product Cards
**Learning:** In React, nested interactive elements (like buttons inside a clickable container div) can cause usability issues where interacting with the inner button also triggers the outer container's action. Additionally, visual star ratings paired with text ratings cause redundant and confusing screen reader announcements if the star images are not explicitly hidden.
**Action:** Always use `onClick={(e) => e.stopPropagation()}` on interactive elements nested within clickable containers to prevent event bubbling. Always use `aria-hidden="true"` on containers of visual ratings and empty `alt=""` attributes on the individual rating images when a text-based equivalent is already present in the UI.
