## 2024-05-18 - Nested Interactive Elements and Redundant Announcements
**Learning:** Adding interactive elements (like wishlist or buy buttons) inside a clickable container (like a product card navigating to details) causes event bubbling, making the user unintentionally navigate away when trying to interact with the button. Additionally, using individual alt text on repeated visual indicators (like star ratings) causes repetitive and unhelpful screen reader announcements, especially when a text-based score is already present.
**Action:** Always add `onClick={(e) => e.stopPropagation()}` to nested interactive elements to prevent triggering parent events. Hide redundant visual indicators from screen readers by adding `aria-hidden="true"` to their container and leaving `alt=""` empty on individual image elements.
