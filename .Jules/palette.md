## 2024-04-10 - ProductCard Event Propagation and Screen Reader Redundancy
**Learning:** In clickable card components like `ProductCard.jsx`, nested interactive elements (like Wishlist or "Buy Now" buttons) will inadvertently trigger the parent card's navigation event. Additionally, rendering 5 individual star icons for a visual rating causes screen readers to redundantly announce "star icon" five times alongside the numerical rating text.
**Action:** Always apply `onClick={(e) => e.stopPropagation()}` to interactive buttons nested within clickable containers to isolate their functionality. Hide purely visual star ratings from screen readers using `aria-hidden="true"` on their parent container, and use empty `alt=""` attributes on the star images when a numerical text rating is already provided.
