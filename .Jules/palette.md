## 2024-04-13 - [Nested interactive elements & star rating accessibility]
**Learning:** Visual star ratings accompanied by a text score (e.g. "4.5") are read redundantly by screen readers. Furthermore, clickable product cards that contain independent interactive buttons (like 'Buy now' or 'Wishlist') trigger the card's navigation event incorrectly when the button is clicked, unless the event propagation is explicitly stopped. This is a severe WCAG violation if the card's outer div acts as a container for other buttons.
**Action:** Always add `aria-hidden="true"` to visual star container elements and set their individual star image `alt` attributes to `alt=""` when there's an accompanying text rating. Always ensure nested interactive buttons within a parent interactive element use `e.stopPropagation()` in their event handlers to avoid triggering multiple actions.
