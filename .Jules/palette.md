## 2024-05-15 - [ProductCard UI elements lacking accessibility]
**Learning:** The application's ProductCard component contains icon-only buttons (like the heart icon for wishlist) that lack an `aria-label`, rendering them inaccessible to screen readers. Additionally, clickable nested elements within the main `div` (like buttons) may inadvertently trigger the parent container's `onClick` navigation due to event bubbling.
**Action:** I will add `aria-label='Add to wishlist'` to the heart button. I will also add `e.stopPropagation()` to the `onClick` handlers of the wishlist and 'Buy now' buttons within the card so they can function independently without triggering a page redirect.
